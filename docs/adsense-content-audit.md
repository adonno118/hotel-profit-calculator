## 7. SEO·색인 점검 결과

- canonical과 `og:url`: 점검 대상 기존 23개 페이지 모두 extensionless 구조이며 `/terms`도 동일하게 추가
- 내부 콘텐츠 링크: `/guide/*.html`, `/examples/*.html`, `/guide.html` 형태의 잔존 링크 없음
- redirect: 각 `.html` 파일에서 extensionless URL로 301 규칙 유지, `/terms.html` 추가
- robots: `User-agent: *`, `Allow: /`; 주요 페이지 차단 없음
- noindex: 주요 페이지에 없고 `index,follow,max-image-preview:large` 사용
- sitemap: extensionless URL만 사용
- 제목/설명: 기존 자동 테스트가 페이지별 중복을 검사함. `/terms`도 고유 메타데이터 사용

`발견됨/크롤링됨 - 현재 색인이 생성되지 않음`은 파일만 고친다고 즉시 해소된다고 보장할 수 없다. 배포 후 URL 검사, 실시간 테스트, sitemap 재제출과 실제 크롤링 대기가 필요하다.
