기능 소개 및 이용 가이드

숙박업 손익 계산, 이렇게 사용하세요

처음에는 간편 분석으로 가정을 시험하고, 검토할 가치가 있는 매물은 상세 분석에서 실제 자료에 맞춰 조정하는 방법이 효율적입니다.

1. 간편 분석으로 시작하기

객실 수와 연면적

객실 수에는 실제로 판매하거나 달방으로 운영할 수 있는 객실을 입력합니다. 공사 중이거나 장기간 사용할 수 없는 객실은 제외하는 편이 안전합니다. 연면적은 건축물대장이나 매물 자료의 ㎡ 값을 입력합니다. 화면에 평 환산값도 표시되며 1평은 약 3.3058㎡로 계산합니다.

객실당 예상 월매출

객실 한 실이 한 달 동안 만드는 평균 매출 가정을 입력합니다. 200만원을 입력하고 객실이 30실이면 예상 월매출은 6,000만원입니다. 빠른 선택값은 입력을 돕는 예시일 뿐 실제 평균이 아닙니다. 성수기 한 달이 아닌 연중 평균에 가까운 보수적인 값을 함께 시험해 보세요.

임대 조건과 직원 비용

보증금과 권리금은 총 초기 투자금에 포함됩니다. 월 임대료는 매월 운영비에 들어갑니다. 직원 수와 1인당 평균 월급을 입력하면 총 급여가 계산되며, 4대보험 등 사업주 부담은 설정된 참고 비율로 별도 추정됩니다. 정확한 법정 부담액은 고용 형태와 보수 등에 따라 달라집니다.

2. 상세 분석으로 조정하기

매출 항목 추가

숙박 매출, 대실 매출, 달방 매출과 기타 매출을 각각 추가할 수 있습니다. 항목명과 월 예상 금액을 직접 수정할 수 있으며 같은 유형의 매출도 여러 줄로 나눌 수 있습니다. 예를 들어 평일 숙박과 주말 숙박을 따로 두면 가정을 검토하기 쉽습니다.

비용 항목 추가

빠른 비용 버튼을 누르면 월세, 급여, 플랫폼 수수료, 공과금과 운영비 항목이 즉시 추가됩니다. 같은 항목을 여러 번 추가할 수 있으므로 급여를 프론트 직원, 청소 직원, 관리자처럼 나누고 이름을 바꿔 사용할 수 있습니다. 불필요한 항목은 오른쪽 삭제 버튼으로 제거합니다.

고정비와 변동비의 차이

고정비는 매출 변화와 관계없이 비교적 일정하게 발생하는 월세, 정규 급여, 보험료, 프로그램 사용료 같은 비용입니다. 변동비는 매출이나 객실 회전이 늘수록 함께 증가하기 쉬운 플랫폼 수수료, 어매니티, 세탁비와 소모품비 등을 말합니다. 구분은 절대적이지 않으므로 실제 계약과 운영 방식에 맞게 변경하세요.

플랫폼 수수료 입력

플랫폼 수수료는 직접 금액 또는 전체 매출 대비 비율로 입력할 수 있습니다. 비율 입력을 선택하면 현재 월 총매출에 비율을 곱해 계산합니다. 플랫폼을 거치지 않는 매출까지 포함하면 비용이 과대 추정될 수 있으므로, 실제 OTA 매출 비중을 알고 있다면 직접 금액으로 보정하는 것이 좋습니다.

공과금 예상 기능

전기·수도·가스 항목은 현재 연면적과 면적당 예시 계수를 곱한 추정값을 적용할 수 있습니다. 연면적이 바뀌어도 직접 수정한 값은 자동으로 덮어쓰지 않습니다. 필요할 때 ‘현재 기준 추정값 적용’을 눌러 다시 계산하세요. 가능하다면 이전 운영자의 최근 12개월 고지서를 확인해 계절 변동까지 반영하는 것이 좋습니다.

투자비용 입력

보증금, 권리금, 시설 인수비, 리모델링비와 기타 초기 투자비를 각각 추가합니다. 현재 계산은 입력한 모든 투자비의 합계를 투자수익률과 회수기간의 분모로 사용합니다. 보증금처럼 회수 가능성이 있는 자금과 소모성 공사비를 구분해 해석할 필요가 있습니다.

3. 주요 결과 해석

영업이익과 영업이익률

월 영업이익은 월 총매출에서 입력한 월 총비용을 뺀 값입니다. 영업이익률은 월 영업이익을 월 총매출로 나눈 비율입니다. 매출이 0이면 비율을 계산하지 않습니다. 금융이자, 세금, 감가상각 등을 비용으로 입력하지 않았다면 결과에도 포함되지 않습니다.

1억 원 투자당 순익

입력한 총 투자비 중 1억 원에 해당하는 월 또는 연 영업이익을 비례 계산한 값입니다. 규모가 다른 매물을 같은 기준으로 비교할 때 참고할 수 있지만, 자금 회수 가능성, 담보 가치, 대출 조건과 위험 차이는 보여주지 않습니다.

예상 투자 회수기간

총 투자비를 연 영업이익으로 나눈 단순 회수기간입니다. 영업손실이거나 이익이 0이면 산정하지 않습니다. 시간가치, 이자, 세금, 추가 공사와 매각가를 반영하지 않은 값이므로 투자 의사결정의 단독 기준으로 사용하면 안 됩니다.

4. 운영방식 비교

숙박 운영, 달방 운영과 하이브리드의 월 매출·비용·이익을 나란히 확인할 수 있습니다. 하이브리드는 숙박 객실과 달방 객실 수를 입력하며 합계가 전체 객실을 초과하면 경고합니다. 비교값은 현재 입력에 설정된 상대 계수를 적용한 시나리오이므로 실제 운영 계획이 있다면 각 매출과 비용을 직접 구성해 다시 계산하세요.

5. 결과를 검증하는 방법

  • 객실당 매출을 낙관·기준·보수 세 값으로 바꿔 봅니다.
  • 월세와 인건비가 예상보다 10~20% 높아지는 경우를 시험합니다.
  • 최근 매출 장부, OTA 정산서와 공과금 고지서를 확인합니다.
  • 시설 점검으로 향후 보수·교체비를 별도 투자비에 반영합니다.
  • 계약 전 세무·노무·법률 조건을 전문가에게 확인합니다.
자동 저장 안내

입력값은 현재 브라우저의 localStorage에 자동 저장됩니다. 다른 기기나 브라우저와 동기화되지 않으며, 브라우저 데이터 삭제 시 함께 사라질 수 있습니다.
